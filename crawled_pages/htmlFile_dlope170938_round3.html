How it work

  1. After purchasing, download the plugin zip file.
  2. Install and activate the plugin like any other plugin from the plugins area in WordPress admin.
  3. Navigate to the page or post editor where you want to write some content.
  4. Configure the Open AI Api Key if you haven’t already from the plugin sidebar
  5. Press “space” after inserting a new paragraph block.
  6. Add your prompt and press “enter“.
  7. Review your generated content
  8. (Optional) rewrite your content if you don’t like a certain part, You can open the rewriting menu by simply highlighting the content you want to rewrite and “right click“.

# CS172

## Usage:
- Make sure you have python3 downloaded and PyLucene available for use
- In your terminal, run the following commands:
  - pip3 install beautifulsoup4
  - pip3 install lxml
  - pip3 install requests
  - pip3 install flask
- To run, use ./indexer.sh
